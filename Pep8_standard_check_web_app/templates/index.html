Правила пользования:

1. Нажмите на область загрузки файла.

2. Выберите файл, который хотите проверить на соответствие стандарту pep8.

3. Нажмите на кнопку "Отправить", чтобы перейти на страницу проверки.

4. Если ваш файл соответствует стандарту, в левом окне вы увидите "Файл соответсвует стандартам pep8". В противном случае там будут выведены причины несоответствия.

5. Если файл не соответствует стандарту, то его можно исправить в правом окне.

6. для повторной проверки файла, нажмите "Проверить".

7. Чтобы вернуться к выбору файла, нажмите "Выбрать файл".
